# TODO App

TODO App is an app for effectively managing tasks.

## Features
- Create, read, update, and delete todos
- Dark theme support
- Support for i18n (English and French)
- Unit and component testing
- GitHub actions running all tests on PR events
- E2E tests for create and update todos

## Stack and Tools
- React with TypeScript
- Jest
- Zustand
- TanStack Query
- React Router
- Tailwind CSS

## Local Setup

Follow the following steps to run the project locally, including running all the tests.

```bash
# Clone the repository
git clone https://github.com/alain-kubwayo/react_technical_assignment.git

# Install the project dependencies
pnpm install

# Spin up the local development server
pnpm run dev

# Run tests
pnpm run test
```
